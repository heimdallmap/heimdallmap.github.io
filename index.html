What is Heimdall?

Heimdall is a desktop developer tool that generates interactive system maps from source code. It uses static analysis to extract classes, structs, interfaces, enums, functions, traits, and modules from codebases, then visualizes them as navigable dependency graphs. Heimdall supports architecture discovery, dependency tracing, code exploration, and usage analysis. It is built for software engineers who need to understand, navigate, and reason about complex software systems. Available for macOS, Windows, and Linux.
