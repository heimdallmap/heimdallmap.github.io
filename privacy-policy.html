Privacy Policy

Version 1.0

HeimdallMap is a local-first code visualization tool: your source code stays on your device, and only telemetry is optionally transmitted to help improve the product.

Data Controller

Heimdall
Contact email: app@heimdallmap.com

What data is collected

HeimdallMap collects telemetry data such as application version, operating system, feature usage statistics, performance metrics, crash reports (if enabled), and basic diagnostic information.

What data is NOT collected

HeimdallMap does not collect, upload, transmit, store, or inspect:

  • Source code
  • Repository contents
  • Git history
  • Credentials, tokens, or passwords
  • AI training data

Purpose of telemetry

Telemetry is used solely to improve application stability, performance, usability, and compatibility across operating systems.

Retention period

Telemetry data is retained for no longer than 12 months unless a longer period is required for security or legal purposes.

Third-party services

Telemetry and crash reporting are processed by our analytics providers acting as a data processor on our behalf.

User rights (GDPR)

You have the right to request access, deletion, objection, or submit a complaint to a supervisory authority regarding your data.
