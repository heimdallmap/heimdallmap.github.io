What never leaves your computer

  • Source code
  • Repository contents
  • Git history
  • Branches and tags
  • Credentials
  • Build artifacts

What may be sent

Telemetry such as app version, OS version, feature usage counts, performance timings, and crash diagnostics.

Telemetry controls

You can easily disable telemetry in the app: Settings → Privacy → Share telemetry.

Security model

Repositories are read directly from the local filesystem. Visualization is generated locally. There is no cloud indexing, no remote code analysis, and no AI processing.

Enterprise considerations

HeimdallMap is designed for use with private and proprietary repositories because source code is processed locally and is not transmitted to HeimdallMap servers. Your IP is fully protected on your own machine.

Updates

The app checks for updates and only basic information like your OS and current app version is sent during that process.
